Эти работы показывают мой базовый frontend-опыт: работу с разметкой, сеткой, отступами, анимациями и адаптивом.

## Технологии

- HTML5 (семантическая разметка).
- SCSS (переменные, вложенность, структура стилей).
- Адаптивная верстка (flexbox, media queries).
- CSS-анимации и переходы.
- Git и GitHub для хранения кода.
- GitHub Pages для публикации статических сайтов.

## Что демонстрируют эти работы

- Понимание структуры макета, сеток и расположения элементов.
- Умение работать с отступами, размерами, типографикой и позиционированием.
- Опыт адаптивной верстки под разные разрешения экрана.
- Использование CSS-анимаций и плавных переходов.
- Развёртывание статических страниц на GitHub Pages.
